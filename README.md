# CrisisLensAI

A Streamlit-based disaster intelligence dashboard scaffold for crisis monitoring, prediction, mapping, analytics, and AI assistance.

## Structure

- app.py: main application entry point
- pages/: Streamlit page modules
- models/: trained model artifacts and encoders
- data/: input and prediction datasets
- utils/: reusable prediction, map, chart, and Firebase helpers
- assets/: logos and regional images

## Getting started

1. Create a Python environment.
2. Install dependencies:
   pip install -r requirements.txt
3. Run the app:
   streamlit run app.py
