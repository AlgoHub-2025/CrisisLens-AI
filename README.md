# 🌍 CrisisLens-AI

### Turning Chaos into Clarity with Real-Time AI Crisis Intelligence

---

## 🚨 Overview

**CrisisLens-AI** is an AI-powered civic intelligence and disaster response platform designed to transform how crisis data is detected, visualized, and acted upon in real time.

In emergency situations, information is often fragmented, delayed, or unreliable. CrisisLens-AI solves this by integrating **real-time data streams, AI-driven analysis, and geospatial visualization** into a single unified dashboard.

Our goal is simple:

> Help responders and citizens understand what is happening, where it is happening, and what to do next.

---

## 💡 Problem Statement

During disasters and emergencies:

* Information is scattered across multiple sources
* Response teams lack real-time situational awareness
* Citizens struggle to access verified updates
* Decision-making becomes slow and reactive instead of proactive

This delay in intelligence can cost lives.

---

## 🚀 Our Solution

CrisisLens-AI provides a centralized platform that:

* 📡 Aggregates real-time crisis and emergency data
* 🧠 Uses AI to interpret and summarize situations
* 🗺️ Visualizes events on interactive maps
* 📊 Displays actionable insights for responders and authorities
* 🔔 Enables faster and more informed decision-making

---

## 🧠 Key Features

* Real-time crisis monitoring dashboard
* AI-powered situation analysis and summarization
* Interactive geospatial visualization using Mapbox
* Live data integration from external APIs
* Clean and responsive UI for all devices
* Modular architecture for scalability and extension

---

## 🛠️ Tech Stack

**Frontend**

* React.js
* Vite
* Tailwind CSS
* Mapbox GL JS

**Backend & APIs**

* Node.js / Express.js
* REST APIs
* Firebase (Auth / Data handling)

**AI & Integrations**

* OpenAI / LLM APIs
* Live data processing pipelines
* External crisis & mapping APIs

**Deployment**

* Vercel (Frontend Hosting)
* AWS / Cloud services (Backend-ready architecture)

---

## 🧩 System Architecture

```
Data Sources → API Layer → AI Processing Engine → Visualization Layer (Map + Dashboard)
                                      ↓
                              Real-Time Insights UI
```
